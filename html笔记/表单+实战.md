# 表单

- 什么是表单？
  - 表单只是一个范围，看不见的，在这个范围内可以设置很多控件

**作用**  用于收集用户信息，进行人机交互操作

**包含元素** 文本框，单选按钮，列表框，图片框，复选框等；这些元素，统称“控件”

# 控将的常用属性

| 属性     | 作用                             |
| -------- | -------------------------------- |
| name     | 指定控件的名称                   |
| id       | 指定标签的唯一标识               |
| value    | 输入（收集，设置）控件的值       |
| checked  | 复选框（单选）组默认被选中的项目 |
| selected | 列表框默认被选中的项目           |
| src      | 图片框的图片来源                 |
| onclick  | 鼠标单击事件                     |
| disabled | 禁用该控件                       |
| multiple | 允许多选（适合普通列表框）       |
